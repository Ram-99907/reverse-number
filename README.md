# Reverse Order of a Number in C

## Description
This program reverses a given integer number.
It is written in C language and suitable for beginners and BTech 1st year students.

## Algorithm---
1. Read the number from the user,
2. Extract the last digit using modulus operator (%).
3. Add the digit to the reversed number.
4. Remove the last digit using division (/).
5. Repeat until the number becomes zero.

## Sample Input
Enter a number:5211

## Sample Output
Reversed number =1125
